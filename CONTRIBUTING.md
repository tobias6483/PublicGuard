# Contributing to PublicGuard

Thanks for helping make PublicGuard better.

## Development

Build the project:

```sh
swift build
```

Run the app:

```sh
swift run PublicGuard
```

## Project Direction

PublicGuard should stay:

- Local-first
- Privacy-first
- User controlled
- Transparent about permissions
- Small enough to audit

## Good First Issues

- Add unit tests for guard state and event logging
- Improve menu bar copy
- Add a configurable grace period
- Add user notifications for silent mode
- Research reliable macOS lid-close detection
- Add network change trigger support

## Pull Requests

Please keep pull requests focused and explain:

- What changed
- Why it changed
- How it was tested
- Whether it changes privacy, permissions, or background behavior
